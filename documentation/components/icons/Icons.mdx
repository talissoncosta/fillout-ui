# Icons

The Icon system is built using a reusable `Svg` component that standardizes size, color, accessibility, and layout. All icons are React components that wrap inline SVG paths and support consistent props.

## Usage

Each icon can be imported and rendered as a standalone component:

```tsx
import { FileTextIcon } from 'src/components/icons'

<FileTextIcon size="small" title="File icon" />

```
