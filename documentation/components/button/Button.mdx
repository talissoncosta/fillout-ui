# Button

The `Button` component provides a flexible and styled button for use across the interface.
It supports a `variant` prop to visually represent states like "active" with customized styles.

## Usage

Use the `variant` prop to control the button's visual behavior:

- `standard`: default state
- `active`: shows a highlighted background, border, and stronger text contrast
