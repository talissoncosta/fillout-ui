# IconButton

`IconButton` is a compact button used to display a single icon, typically for secondary actions. It supports an optional `tooltip` prop for additional context, shown on hover or focus.


---

## Default usage

Use `IconButton` to represent common UI actions like "info", "delete", or "edit" using icons only.

## Accessibility

- Tooltip appears on hover and keyboard focus.
- Button element is focusable and uses semantic HTML.
- Icon should include an `aria-label` or visually hidden text if used without tooltip.

---

## Best Practices

- Use for icon-only actions
- Avoid using multiple `IconButton`s too close together
- Provide a `tooltip` or `aria-label` for accessibility
