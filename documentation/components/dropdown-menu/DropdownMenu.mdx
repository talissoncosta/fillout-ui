# DropdownMenu

The `DropdownMenu` component is a custom menu built using [@radix-ui/react-dropdown-menu](https://www.radix-ui.com/docs/primitives/components/dropdown-menu).
It is composed of trigger and content primitives, offering full control over layout, animation, accessibility, and keyboard navigation.

Use it to render contextual actions, such as page-level settings or inline UI operations.

---

## Usage

Wrap `DropdownMenuTrigger` around any clickable element (usually a `Button`).
Use `DropdownMenuContent` to provide your menu structure, including optional titles, separators, and icon-labeled items.

<Canvas of="Default" />

---

## Anatomy

- `DropdownMenu`: Root context provider.
- `DropdownMenuTrigger`: The element that opens the menu.
- `DropdownMenuContent`: The floating panel containing the menu options.
- `DropdownMenuItem`: Clickable option inside the menu.
- `DropdownMenuSeparator`: Horizontal divider between groups of items.

---

## Accessibility

This component handles:

- Keyboard support (arrow keys, `Enter`, `Escape`)
- Focus trapping and loop navigation
- ARIA roles (`menu`, `menuitem`)
- Screen reader-friendly structure
